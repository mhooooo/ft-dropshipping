Key Design Decisions

  • SQLite over JSON — 14 indexed tables for relational queries
  • Platform fees baked in — Lazada/Shopee/TikTok commissions auto-applied
  • Thai ×9 pricing — round_thai_price() for ฿389/฿599 format
  • CJ → THB auto-conversion — configurable USD rate (default 36.5)
